Simulation Framework
• 10,000 Monte Carlo paths simulated across the selected time horizon
• Annual contributions added at the end of each period
Return Modeling
• Parametric scenarios assume normally distributed returns (μ, σ)
• Bootstrap scenario resamples actual S&P 500 returns (1928–2025)
• Captures historical volatility, drawdowns, and fat-tail behavior
Inflation Adjustment
• Constant annual inflation rate applied to compute real (today’s dollar) values
• Real values represent purchasing power at the end of the simulation horizon
Limitations
• Does not model taxes, fees, or account-specific withdrawal rules
• Assumes constant contribution behavior and no market timing
• Results are probabilistic estimates, not guarantees of future performance
